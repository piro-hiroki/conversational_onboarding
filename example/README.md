# example

Runnable demos for [`conversational_onboarding`](../README.md).

```bash
cd example
flutter pub get
flutter run
```

Three templates are included on the home screen:

- **Cal AI** — health / fitness with analyzing-your-plan reveal and
  branching by experience level.
- **Duolingo** — learning app, grid-style choices, level + goal.
- **Headspace** — wellness, ranking step, permission priming.

Each template is ~50 lines of code — see `lib/templates/`.
